# videoCollector

多平台聚合视频采集器，可根据自定义条件采集全网视频

## 配置参数的说明
下载最右视频，需要先对手机进行抓包
下载youtube 需要填写自己的socket5代理地址（等后续版本不需要此项）


## 使用

下载压缩包解压

填写配置文件后

双加打开 `vc.exe` 即可
